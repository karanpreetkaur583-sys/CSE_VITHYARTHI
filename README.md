# CSE_VITHYARTHI
This file implements a simple text-based Library Management System using Python. It allows users to add, view, borrow, return, and search for books stored in a local file named "books.txt". The system presents a menu-driven interface and manages each operation with dedicated functions.
imple Library Management System (Console-Based)

A clean, beginner-friendly Python program to manage a small library.
 Features

- Add new books (Title + Author)
- View all books with status
- Borrow a book
- Return a borrowed book
- Search books by title
- All data saved automatically in books.txt
- Simple and easy-to-use menu

1. Make sure you have Python 3 installed
2. Save the Python code as library.py (or any name you like)
3. Open terminal / command prompt in the folder
4. Run the program:


python library.py
That's it! The file books.txt will be created automatically to store your books.
